React Challenge Tracker is a mini project built using React that allows users to add personal challenges and track their progress. Each challenge can be marked as completed or failed, with smooth animations enhancing the user experience.

Features:
- Add new challenges
- Mark challenges as Completed or Failed
- Animated UI for better interaction
- Simple and clean component-based structure

This project was created as a learning exercise to practice React fundamentals and UI animations.

Tech Stack:
- React
- CSS Animations / Framer Motion (if used)
- JavaScript (ES6)
